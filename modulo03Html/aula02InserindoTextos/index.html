Texto

A tag "p" é usada para escrever texto como esse que você esta lendo

Para adicionar negrito em uma palavra deve ser usada a tag "strong"

Para adicionar italico em uma palavra deve ser usada a tag "em"

Para adicionar marca texto em uma palavra deve ser usada a tag "mark"

Para riscar o texto deve ser usada a tag "del"

Para deixar o texto pequeno deve ser usada a tag "small"

"span" é uma tag de texto muito util para quando precisar estilizar uma parte especifica do texto com css, ou quando vou adicionar texto via javascript

Links

Para adicionar um link deve usar a tag "a"
